# CalorEasy 🍎

## Description
CalorEasy is a responsive React web application that allows users to track daily calorie intake and macronutrients. Users can search for foods using an external nutrition API, add them to their tracker, and visualize nutrition breakdowns with interactive charts.

## Features
- Manual food entry (name, calories, protein, carbs, fat)
- Search food database via API
- Track total calories
- View nutrition breakdown charts (Protein, Carbs, Fat)
- Dark / Light mode toggle
- Responsive design

## How to Run
1. Clone the repository: `git clone https://github.com/username/caloreasy.git`
2. Install dependencies: `npm install`
3. Run the app: `npm start`
4. Open http://localhost:3000 in your browser
